# 2026-06-27 今日项目决策

## 选择

今天做 `Brain Nutrient Map Web`，中文名为「脑健康成分地图」。

## 为什么不是体检单

用户明确表示不必限定在体检单。最新线上 AI 延续学商机日报中，传播性最强的机会是「麦角硫因 × 中年脑健康」，它比专业衰老时钟代码库更适合当天快速做成可分享项目。

## 为什么不是 HRT 女性激素项目

女性激素与脑老化研究受众清晰，但合规边界更复杂。HRT 不能做推荐或对比工具，今天只作为背景证据，不作为主项目。

## 为什么不是表观遗传衰老时钟 GitHub 项目

`congca/Environmental-Sociodemographic-and-HIV-Correlates-Across-Epigenetic-Aging-Clocks` 更适合做专业资料库或教程，不适合当天做大众传播入口。它可以作为后续「衰老时钟导航页」项目。
